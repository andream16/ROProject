# ROProject
Symmetric CVRPB solver for any TSP instance.

## What you need to run it

 - JDK : v1.8
 - Commons-io : v2.5
 - JFXRT : v1.8
 
Both of the jars can be found inside the project folder.

## How to import the libraries with IntelliJ Idea

File > Project Structure > Libraries > Click on Plus Button > Chose Java > Link the path of the JARS > Ok.

## How you can Execute it

Run `Euristica2Fasi.java`. To select a given instance, please, change line `31` containing `String FILE` to the name of an instance, for instance `A1`.

## How it works
After running it on an instance, it produces results inside `Results` folder.
